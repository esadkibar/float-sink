# Float–Sink (Washability) Table Calculator

Web-based tool for coal preparation engineers.

## Features
- Automatic float–sink table calculation
- Cumulative float & sink curves
- Elementary ash curve
- Density curve with top density axis
- Single combined chart (industry-standard layout)
- CSV export

## Input
Only:
- Mass percentage (%M)
- Ash percentage (%K)

All other columns and curves are calculated automatically.

Built for coal washability analysis.
